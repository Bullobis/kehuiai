## 开发流程

### 1. 报名
在本帖子下留言「我要报名 + 岗位」

### 2. 分配
我(kuleilu)会：
- 发送源代码压缩包
- 分配具体任务
- 说明技术要求

### 3. 开发
- 从 GitHub 拉取最新代码
- 在分配的任务分支开发
- 定期同步进度

### 4. 提交
- 提交 Pull Request
- 我审核代码
- 合并到主分支

### 5. 测试
- APK 打包测试
- 真机兼容性测试
- 发布新版本
